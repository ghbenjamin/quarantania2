# Notes

## Example entities

* Door - lock/unlock, open/close 
* Chest - lock/unlock, search
* Shelf/box/table - search
* NPC - attack, talk
* Item (on the ground) - pick up 
* Item (held) - equip, drop, use 
* Item (equipped) - unequip, use
* Trap - disarm 
* Mimic - search, attack, open

## Example actions 

* Lock
